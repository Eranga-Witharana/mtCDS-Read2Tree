# Read2Tree Reconstruction

## Software Installation

Create and activate a dedicated conda environment:

```bash
conda create -n r2t python=3.10
conda activate r2t
```

Install Read2Tree and required dependencies according to the official installation instructions.

## Verify Installation

```bash
read2tree --help
```

## Input Data

Required inputs:

* Amino acid reference database (`ogs_aa`)
* Nucleotide reference database (`all_mt_cds.fasta`)
* Filtered paired-end sequencing reads

## Step 1: Marker Preparation

Prepare orthologous group markers from the amino acid reference database.


```bash
read2tree  --step 1marker  --standalone_path path_to_ogs_aa/  --dna_reference path_to_all_mt_cds.fasta --output_path Output_directory --debug   
```

## Step 2: Read Mapping and Sequence Reconstruction

Map filtered sequencing reads to the reference markers and reconstruct mitochondrial CDS 

```bash
read2tree --step 2map --standalone_path path_to_ogs_aa/  --dna_reference path_to_all_mt_cds.fasta --reads path_to_filtered_reads/filtered_Species_A_1.fastq.gz path_to_filtered_reads/filtered_Species_A_2.fastq.gz --output_path Output_directory --debug
```

## Step 3: Alignment Combination

Combine reconstructed sequences into individual and concatenated alignments.

```bash
read2tree  --step 3combine --standalone_path path_to_ogs_aa/  --dna_reference path_to_all_mt_cds.fasta --output_path Output_directory --debug
```

## Output Files

The primary outputs generated by Read2Tree include:

- `concat_merge_dna.phy` – concatenated nucleotide alignment of all recovered mitochondrial CDS
- `06_align_merge_dna/` – individual mitochondrial CDS alignments
- Reconstructed mitochondrial CDS sequences for each target species

The alignments contained in `06_align_merge_dna/` can be used for locus-specific analyses, while `concat_merge_dna.phy` can be used for downstream phylogenetic reconstruction.
